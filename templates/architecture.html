Client Layer

Users access DocCollab through a web browser. The client layer contains the Login Page, Dashboard and Collaborative Editor where users create, edit and manage documents.

Application Layer

The Flask Application Server handles routing, authentication, document management, sharing, comments, version control and real-time collaboration requests.

Real-Time Collaboration Layer

SocketIO enables multiple users to work on the same document simultaneously. Changes are synchronized between connected clients in real time.

Conflict Resolution Layer

When two users edit the same document, the Conflict Resolution Engine applies Operational Transformation concepts and Last Write Wins logic to maintain consistency.

Version Control Layer

Every save operation creates a version entry. Users can review document history and restore previous versions if required.

Storage Layer

SQLite stores documents, comments, version history and collaborator information. The database ensures persistent storage and efficient retrieval.
